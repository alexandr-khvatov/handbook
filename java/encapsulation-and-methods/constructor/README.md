# Constructor

Уровень доступа дефолтного конструктора совпадает с уровнем доступа класса.

```java
class A {
    
}

public class B {
    
}
```

ДЕФОЛТНЫЕ КОНСТРУКТОРЫ:

```java
class A {  // default

    A(){   // default
        
    }
}

public class B { // public
    
   public B(){   // public
        
    }

}
```
