# NESTED CLASSES

![](<../.gitbook/assets/image (139).png>)

Язык программирования Java позволяет определить класс внутри другого класса. Такой класс называется вложенным классом и показан здесь:

```java
class OuterClass {
    ...
    class InnerClass {
        ...
    }
    static class StaticNestedClass {
        ...
    }
}
```
